Configparameters

This is your form description. Click here to edit.

  • Change this to your websever or local file storage if needed

  • Change the value to your current IP of the webRotatorInterface like 192.168.1.120

  • The Time, how often the interface will be queried while rotation to get values (>1000!!)

  • The standard interval time, how often the webRotatorInterface is queried liek a watchdog to get headings or changes

  • How often will the cleanup function try to find some changes due to oeverrunning, wind or remote action. (>30000)

  • If during the CleanupStandardInterval no changes detected, it does not cleanup the Positions. Only after a long time like 15min (> 900000)

  • Define your own Keyboardshortcuts and to what heading they shd point. Copy this into an editor, edit to your needs and put it back. Dont forget to save it in your editor for later use...

  • Only change this if you want to get debug info to find some errors


Short Manual how to use and configure the client

Configparameters

  • URL: This is the rare URI to the location of the rotator.html. This noramly hosted at http://h.mmmedia-online.de/webRotator/rotator.html.
    Dont change it, unless you only want to host your file on your own server (URL) OR save it on your local disk (URL to File, e.g. file:///C:/Users/mmmedia/Google Drive/Funk/Software/RotateMe/rotator.html)

  • IP: This is the IP adress, how your own webRotatorInterface can be reached. This can be a local IP like 192.168.1.120, this can be a global one like 188.194.221.12:88 or some DynDNS entry like dm5xx.dyndns.org:88
    Keep in mind, if you are using global or DynDNS (to call your webRotatorInterface remote), Portforwarding has to be set up in your internet-router!

  • StandardIntervalRotating (in MS): is the time between each looping request to the webRotatorInterface to get the current heading while rotation

  • StandardIntervalTime (in MS): is the continuos time the webRotatorInterface is queried to get the current heading.

  • CeanupStandardIntervalTime (in MS): is the time the webRotatorInterface is queried to get the current heading.

  • CleanupLongRunIntervalTime (in MS):

  • MakroDefinition: Keyboard Shortcuts keyname:heading

  • Debug: true or false

Use the Client

  • First usage: Remember your generated URL to the client. Its still in your clipboard. Open a new Tab in your browser and paste the URL with cntrl+v into the adress line.
    Call the page by pressing Enter. If all is loaded correct, you will see the heading rose and some initialisation. Bookmark the url - the configuration from the configuration page will not be saved!
